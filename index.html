Some Projects

GTD Task Manager

Distributed task manager using several technologies and interoperability methods. Both backend and frontend are uncoupled and made in Java using JSF, PrimeFaces and EJBs. There are also client implementations on C# (CLI) and Android (GUI), interacting with the backend by means of REST (Android) and SOAP (C#). Furthermore, four more CLI clients were developed with Java using REST, SOAP, messaging with JMS, and just EJBs, respectively.

RIP Necrodancer

RIP Necrodancer is a very simple clone game inspired by Crypt of the Necrodancer and made with Cocos2D JS. Bear in mind that it's just an unfinished prototype. Controls: Arrow keys for moving, WASD for attacking (yeah, I could've done better). You can move or make actions following the beat (so try to press the keys in the appropiate time!). You can also place bombs with the space bar.

Try it out

Infinite Sense

Infinite Sense is an infinite runner mobile game implemented using pure Android. You can jump, crouch and hit depending on the region of the screen you tap on (top-left = jump, bottom-left = crouch, right = hit). It's just an experiment, so it's pretty short and simple. New maps can be defined easily in normal txt files where different characters represent different tiles.
